get /api/nodejs
Search for repositories with query 'nodejs' from Github, at first page (apiNodejsGet)
Send a request to Github api to retrive repositories with query 'nodejs' from Github. This endpoint returns 10 items for each call. This request is equivalent to calling 'GET /api/nodejs/1'

Return type

NodeJSRequest

Example data

Content-Type: application/json
{
  "total_count" : 0.8008281904610115,
  "link" : {
    "next" : "next",
    "last" : "last",
    "prev" : "prev",
    "first" : "first"
  },
  "incomplete_results" : true,
  "items" : ""
}

  • application/json
